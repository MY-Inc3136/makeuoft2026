# makeuoft2026
Scavenging robot that detects objects and marks them on a graphical map.


The robot will detect areas using an ultrasonic sensor.
When the robot detects an area, it wil move towards that area and marks its location on a map. 

### Key Functions of the Robot
- Determine distance to a point of interest
- Move towards the point of interest
- Track the location of the point
- Output the location
- Repeat

### Map
The map is an image generated using the points of interest. The points of interest will then be indicated using markers.
